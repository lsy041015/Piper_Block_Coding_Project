홈 캘리브레이션이란?

각 관절(J1~J6)의 홈(원점) 위치를 원하는 각도로 정의하고 저장합니다.
저장 후에는 IDE의 🏠 홈 위치로 이동 블록이 이 위치로 이동합니다.
로봇 전원 켤 때마다 [ 홈으로 이동 ] 버튼 한 번으로 반복 초기화가 가능합니다.

⚡ 빠른 초기화

저장된 캘리브레이션 값으로 로봇을 즉시 홈 위치로 이동합니다.
매 세션 시작 시 이 버튼 하나로 반복 초기화를 수행하세요.

🦾 관절 각도 설정 (−175° ~ +175°)

로봇 드라이버 영점 설정이란?

현재 로봇의 물리적 위치를 해당 관절의 0° 기준점으로 모터 드라이버 내부에 저장합니다.
PC가 아닌 로봇 자체에 저장되며, 어느 PC로 연결해도 동일하게 적용됩니다.
설정 후 로봇을 재시작해야 적용됩니다.

🎯 영점으로 설정할 관절 선택

⚠️ 실행 중 로봇이 잠시 비활성화(Disable)된 후 영점 명령을 전송하고 재활성화합니다.

마스터 / 슬레이브 모드란?

두 대의 Piper 로봇을 연결해 한 대로 다른 한 대를 직접 교시하는 모드입니다.
마스터(교시 입력): 손으로 직접 움직여 동작을 입력하는 팔
슬레이브(운동 출력): 마스터의 움직임을 따라 동작하는 팔
일반 모드: 독립 제어 (마스터/슬레이브 해제)
